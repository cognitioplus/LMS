**Part III: Core Support Skills and Crisis Response**
3.1 Core Support Skills (Module 4)
These eight core skills help you LISTEN effectively and foster self-efficacy 29:
1.	Building Rapport: Establishing trust.
2.	Showing Empathy: Acknowledging their feelings.
3.	Reflecting: Paraphrasing the client's words.
4.	Labeling Emotions: Helping them name what they are feeling.
5.	Asking the Right Questions: Using open-ended questions.
6.	Giving No Advice: Guiding them to their own solutions.
7.	Identifying Strengths: Pointing out their resilience30.
8.	Generative Listening: Deep, non-judgmental focus.
3.2 Recognizing and Responding to Crisis (Module 5)
